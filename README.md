# TASK MANAGER

## 📌 Description
A web-based task management system using Java, Servlets, and MySQL.

## 🛠️ Technologies Used
- Java (Servlets)
- HTML, CSS, JavaScript
- MySQL
- Apache Tomcat

## 🚀 Features
- Add tasks
- Update tasks
- Delete tasks
- User-friendly interface

## ▶️ How to Run
1. Start Apache Tomcat
2. Open browser: http://localhost:8080/TASKMANAGER
